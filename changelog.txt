# Lavender Messenger — Changelog

## 1.0.7.1
- Разделение репозиториев: сервер и Android клиент теперь отдельные git-репозитории
- Сервер: удаление мусора (.DS_Store, .idea, бинари), обновление документации
- Android: обновление README, добавление changelog.txt в .gitignore в будущих версиях

## 1.0.6.34 - UI Perfection
- Унифицированные виджеты: все диалоги как красивые Bottom Sheet
- Современные кнопки: полупрозрачный фон, без Caps-lock, единые 28dp скругления
- Совершенное расстояние и стили для всех интерактивных компонентов

## 1.0.6.33 - Performance & UX
- Оптимизация информации о группах с RecyclerView
- Единые стили кнопок и улучшенный layout настроек
- Темизированные курсоры и иконки в поиске
- Быстрый доступ к профилю по тапу на заголовок чата

## 1.0.6.32 - Interactive Updates
- Пользователь может отменить или продолжить загрузку update через индикатор
- Улучшенная стабильность UpdateManager с возможностью cancelDownload

## 1.0.6.31 - Unified Widget System
- WidgetSystem.kt: StandardBottomSheet, ActionBottomSheet, SearchableListBottomSheet
- Миграция Auth flow и Settings на Widget System
- Глубокая темизация: курсоры, хайлайты, инпуты, индикаторы загрузки
