# Lavender Messenger — Что нового

## 1.1.0.0
- Обнаружение серверов: добавлен публичный RPC ListServers — клиенты автоматически получают список серверов
- Удалён дублирующий GetServers, оставлен единый ListServers (без авторизации)
- API управления сервером: новые gRPC методы для супер-админа (создание/удаление серверов)
- Исправлен тип creator_id на UUID, добавлен creatorUsername
- Улучшена документация proto (messenger.proto + server.proto)
- Удалён встроенный APK-сервер — APK теперь раздаётся через nginx

## 1.0.7.1
- Страница обновлений: теперь можно скачать актуальную версию прямо с сайта
- Исправлены мелкие баги и улучшена стабильность

## 1.0.6.34
- Все диалоги переделаны на современные Bottom Sheet
- Унифицированные кнопки: полупрозрачный фон, мягкие скругления
- Исправлены отступы и стили по всему приложению

## 1.0.6.33
- Ускорена загрузка информации о группах
- Единый стиль кнопок в настройках
- Поиск теперь с темизированными курсорами и иконками
- Тап по заголовку чата открывает профиль

## 1.0.6.32
- Можно отменить загрузку обновления прямо во время скачивания

## 1.0.6.31
- Новая система виджетов: единый стиль для всех окон и меню
- Улучшенная темизация: курсоры, поля ввода и индикаторы адаптируются под тему
